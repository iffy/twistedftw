IRC Bot

Twisted supports making both IRC clients and servers. It's easy (and fun) to make an IRC bot. The bot below is based on the ircLogBot.py example from the official Twisted Words code examples page.

Reminder bot

When this IRC bot is in the room, you if you say T-5, the bot will message you BOOM! after 5 seconds. You can also have the bot message you at a certain time with @2:30 give me the message.

Save the following as a file named annoybot.py: 





  Then join the #twistedftw channel on freenode (this sometimes takes 10 seconds or longer):




python annoybot.py tcp:irc.freenode.net:6667 ${USER}_annoybot twistedftw
