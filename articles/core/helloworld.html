Here is some runnable Python using the Twisted library: 

from twisted.internet.task import react, deferLater

def printMessage():
    print 'Hello, world!'

def main(reactor):
    return deferLater(reactor, 1, printMessage)

react(main)
If you run this program, it will wait 1 second, print Hello, world!, then exit with exit code 0
