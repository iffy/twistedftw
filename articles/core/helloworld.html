Here is some code you can run to make sure you have Twisted installed correctly. Start a python console session (by typing python in your terminal) then paste the following in the console: 

from twisted.internet.task import react, deferLater

def printMessage():
    print 'Hello, world!'

def main(reactor):
    return deferLater(reactor, 1, printMessage)

react(main)
The program will wait 1 second, print Hello, world!, then exit with exit code 0.
